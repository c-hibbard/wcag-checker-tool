🧪 WCAG 2.2 Accessibility Checker

A lightweight, browser-based tool that scans webpages for common WCAG 2.2 accessibility issues. No install, no account, no fluff.

🚀 Try It Live

👉 Run demo now

👉 Or drag this bookmarklet to your bookmarks bar:
WCAG Checker

👉 Use the Chrome Extension: View on GitHub

🔍 What It Checks

  • ❌ Missing alt attributes on images
  • ⚠️ Low contrast between text and background
  • ❌ Form inputs missing associated <label> elements
  • ⚠️ Interactive elements missing focus or keyboard traps
  • ❌ ARIA role misuse

🛠 Extra Tools

🎨 Color Contrast Tester
Check and tweak foreground/background colors to meet WCAG contrast ratios.
🔗 Focus Order Visualizer
Displays keyboard tab order for links, buttons, and form fields.
🏷 ARIA Landmarks Viewer
Outlines regions like <main>, <nav>, <aside>.

📖 How To Use

  1. Open any webpage you want to check.
  2. Click the WCAG Checker bookmarklet or the Chrome Extension button.
  3. View highlighted issues and use toggles to hide distracting elements.
